My Site

Welcome to my journey in learning front end.

Here's what I've built so far:

  • 1. Hypergeometric calculator with graphical comparison of probabilities after changing deck size, hand size, or the number of starters.
  • 2. Grid-style inputs for 1 or 2-card combinations. 0 means brick. 1 means live hand.
  • 3. Advanced probability calculator that supports any size combinations and flexible "types" that map to multiple cards.
  • 4. Tree visualization of combo paths using Graphviz.
