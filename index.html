My Site

Welcome to the first edition of my opening hand calculator.

I'm hoping this develops into an elaborate project with the following functionalities:

  • 1. Allow users to input multiple cards that represent a deck.
  • 2. Perform hand probability calculations using my own algorithm based around integer composition and combinations. This means no simulations or approximations.
  • 3. Show a graphical representation of the results.
  • 4. Compare results between a current run vs. a previous run. Allow user to revert to a previous run.
  • 5. Allow users to input a 2-card combination table of live vs. brick hands.
  • 6. Allow further categorization of the live hands by tiers.
  • 7. Support 3-card hands.
  • 8. Allow drag and drop to quickly define new hands and tiers.
  • 9. Allow runs to be exported and shared based on an url.
  • 10. Support colors.
